Introduction

This documentation provides a comprehensive guide for the Django Assignment provided by ByteQuest, which aims to deploy a CRUD API for a product management system of an ECommerce shopping website. The application utilizes Python, MySQL, Django, Django Rest Framework, and is hosted on PythonAnywhere.

Technologies Used

  • Python
  • MySQL
  • Django
  • Django REST Framework
  • PythonAnywhere
  • Git
  • Docker

Task Overview

The task involves creating a CRUD API for managing products in an ECommerce platform. The codebase is maintained on a public GitHub repository. The main objectives are as follows:
  • Implement CRUD operations for products.
  • Utilize Django models for database interactions.
  • Host the application on PythonAnywhere to make it accessible via HTTP/HTTPS.
  • Utilise the File Sytem/redis for cache
  • Implement the Pytest for the testing the Application using fixture
  • Dockerizing the App

CRUD API Implementation

The CRUD operations are implemented using Django Rest Framework. Each operation corresponds to an API endpoint:
  • Create: POST /products/
  • Read: 
                        Single Product: GET /products/{id}/
                    List of Products: GET /products/
  • Update: PUT /products/{id}/
  • Delete: DELETE /products/{id}/

Database Connection

The application uses MySQL as the database backend. Configuration details can be found in developement.py: 
                DATABASES = {
                    'default': {
                        'ENGINE': 'django.db.backends.mysql',
                        'NAME': 'database_name_you_create',
                        'HOST': 'localhost',
                        'USER': 'mysql_username',
                        'PASSWORD': 'mysql_password'
                    }
                }
                

                CACHES = {
                     'default': {
                         'BACKEND': 'django_redis.cache.RedisCache',
                         'LOCATION': 'redis://localhost:6379/2',
                         'TIMEOUT': 20,
                         'OPTIONS': {
                             'CLIENT_CLASS': 'django_redis.client.DefaultClient',
                         }
                     }
                }
